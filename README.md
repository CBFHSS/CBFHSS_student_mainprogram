# BFHSS_student_mainprogram
CBFHSS 學生端程式負責接收的部分。

## Prerequisites
```bash=
$ pip3 install -r requirements.txt 
```
## Deployment
```bash=
$ python3 manage.py runserver 0.0.0.0:8000
```
